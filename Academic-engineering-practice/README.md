# 我的第一个暑期实践小项目

## 项目用途：药品管理

### 使用语言：C++

### 开发系统：MacOS 10.15 & CentOS 7

### 产品功能：

1. **设置及修改密码**
2. **录入药品基本信息**
3. **药品的入库管理**
4. **药品的销售管理**
5. **修改药品基本信息（包括增加、删除功能）**
6. **输出全部药品基本信息、药品交易信息**
7. **查询管理**
8. **统计管理**
9. **汇总报表**
10. **退出系统**
